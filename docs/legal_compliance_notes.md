# Юридичні вимоги — фіскалізація (ПРРО)

## Обов'язок застосування ПРРО

ФОП 2-ї групи, не платник ПДВ, **зобов'язаний застосовувати програмний реєстратор
розрахункових операцій (ПРРО)** при кожному розрахунку, що проходить через Prom.ua
та Rozetka — незалежно від способу оплати:

- онлайн-оплата (передоплата на рахунок/картку),
- накладений платіж (оплата при отриманні на Новій Пошті).

Це **юридичний обов'язок, а не опція**. На кожен завершений розрахунок має бути
сформований фіскальний чек через ПРРО. Відсутність чека — підстава для штрафу
при перевірці (ризик значно вищий саме через модель "маркетплейс + накладений
платіж", де про обов'язковість чека часто помилково забувають, вважаючи накладений
платіж "готівкою поштового оператора", а не розрахунком продавця).

**Це нотатки для розробки автоматизації, а не юридична консультація.** Перед
запуском регулярних продажів — звірити ці вимоги (конкретний закон, ставки,
винятки за групою ЄП) з бухгалтером/податковим консультантом.
